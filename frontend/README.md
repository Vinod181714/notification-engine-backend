# Frontend for Notification Prioritization Engine

Next.js frontend (minimal scaffold) — runs on port 3001 by default.

Install & run:

```bash
cd frontend
npm install
npm run dev
```

Environment:
- `NEXT_PUBLIC_API_URL` (defaults to `http://localhost:3000`)

Screens included:
- Login
- Events list & detail
- Rules list & create
- Metrics
- Health
- Later Queue

This is a minimal scaffold; extend styling and authentication handling as needed.
